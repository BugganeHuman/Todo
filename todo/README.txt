ToDo App
A simple ToDo application built with HTML, CSS, and JavaScript using localStorage to save tasks. Open-source 

 Important Notice

This ToDo uses localStorage, which means anyone who has access to your device, or malicious software, can view your tasks.
Do not store passwords, personal data you are not willing to share, or any other confidential information here.
This project is open-source and available on GitHub. You can modify the code and deploy it yourself if needed.

Features

Add new tasks using the input field.
Mark tasks as completed with a checkbox.
Completed tasks are moved to a separate Completed Tasks page.
Supports light and dark themes.
All tasks are saved in localStorage for persistence.
Open-source and fully customizable.

How to Use

Open index.html in your browser.
Enter a task in the input field and press Enter or click Add.
To complete a task, check the box next to it. It will move to the Completed Tasks page.
